📌 Project Title:

Fraction Division – Visual Learning Game

🎯 Project Description:

This is an interactive web-based educational game designed to help students understand fraction division using visual interpretation.

The game uses:

🍕 Pizza model (visual fractions)
📏 Number line representation
🔄 Step-by-step method (Keep · Change · Flip)

It makes learning fun, interactive, and intuitive.

🚀 Features:

🎮 Interactive quiz-based gameplay
🎨 Beautiful UI with animations and effects
📊 Score, streak, and XP tracking
🧠 Step-by-step explanation of each question
💡 Hint system for better understanding
🎯 Multiple difficulty levels:
Easy (Unit fractions)
Medium (Proper fractions)
Hard (Mixed & Improper fractions)
🔊 Sound effects and feedback
🧩 Visual learning with pizza slices & number line
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling, animations, responsive design
JavaScript (Vanilla JS) – Game logic & interactivity

▶️ How to Run the Project:

🔹 Method 1: Using Browser

🎮 How to Play:

Select a difficulty level
Click Start Adventure
Solve fraction division questions
Use visuals (pizza & number line) to understand
Choose the correct answer
Earn points, streaks, and XP

📚 Learning Concept Used:

Fraction Division Rule.

👉 Keep · Change · Flip

Example:

a/b ÷ c/d = a/b × d/c

The game visually demonstrates:

How many times one fraction fits into another.
Step-by-step multiplication after flipping.
